# 📧 Spam Detector

A machine learning project that detects spam messages with 98% accuracy.

## 🤖 Models Used
- Naive Bayes
- SVM (Support Vector Machine)

## 📊 Results
- Accuracy: 98%
- Visualizations: Confusion Matrix & ROC Curves

## 📁 Files
- `Untitled.ipynb` - Main notebook with full code
- `create_dataset.py` - Dataset preparation
- `verify_dataset.py` - Dataset verification

## 🛠️ How to Run
1. Install requirements: `pip install scikit-learn pandas numpy matplotlib seaborn`
2. Open `Untitled.ipynb` in Jupyter
3. Run all cells
